== Installation ==

You need:
* WordPress Version: 4.7 or higher
* an active PantherMedia Account with Credits

How to:
* Log into WordPress
* Under Plugins search for "PantherMedia"
* Install the Plugin

ATTENTION: The Plugin requires the following PHP extensions:
* JSON
